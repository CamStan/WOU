PROJECT TITLE: CS162 Final Project "Book Manager"

PURPOSE OF PROJECT: For a book auther to track his/her daily word counts
and time spent on each of their projects. Primarily used for tracking
productivity and highlighting patterns.

VERSION or DATE: Version 1.0 - 6/6/2014

HOW TO START THIS PROJECT: The main() method inside the BookManager class.

AUTHORS: Cameron Stanavige

USER INSTRUCTIONS: Start the program using the main() method of the
BookManager class. The instructions will be displayed in a popup menu.
They can be accessed at any time from the 'Help' menu by clicking the 
'View Help' option.
